3、安装docker和docker-compose

安装指令，如下：

Centos :

​ yum update -y #更新数据源信息

​ yum install -y docker # 安装docker

​ yum install -y docker-compose # 安装docker-compose

安装完成！！

#启动docker

systemctl start docker

#设置开机自启

systemctl enable docker

确保docker镜像，运行程序为空，不为空的话，网上自己寻找删除教程：

4、加载ARL离线镜像

加载ARL离线镜像：

​ docker load < arl_2.6.1.tar

加载数据库的离线镜像：

​ docker load < arl_mongo.tar

​ docker load < arl_rabbitmq.tar

解压成功之后查看镜像，应该与我的是一致的，如下：

解压ARL-2.6.1.zip文件，如下：

修改ARL-2.6.1/docker/.env的内容，如下：

vi ARL-2.6.1/docker/.env

修改为 v2.6.1 esc + :wp 保存退出

修改成功

首次部署需要执行：

​ docker volume create arl_db

进入ARL-2.6.1/docker中去创建容器：

​ cd ARL-2.6.1/docker

创建和启动容器：

​ docker-compose up -d

灯塔地址：https://IP:5003 (vps中记得要去在防火墙开放5003，本地的入站规则也要开启5003端口)

搭建成功，初始账号/密码：admin/arlpass
